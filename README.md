# Music Player Setup

## Windows MPV DLLs

To run this application on Windows, you need the MPV DLLs. Download the required files from:

https://github.com/shinchiro/mpv-winbuild-cmake/releases/download/20260421/mpv-dev-x86_64-v3-20260421-git-5921fe5.7z

Extract the DLLs (such as `libmpv-2.dll`) to your project root or a directory included in your PATH.
